# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0.0] - 2025-12-11

### Added
- Initial release of the Functorial Fission library
- Core module with Nuclide and NuclearConfig types
- SEMF module implementing the Semi-Empirical Mass Formula
- Functor module with binding energy functor and morphisms
- Coalgebra module for chain reaction dynamics
- Reactor module for practical power calculations
- TypeSafe module demonstrating compile-time conservation law enforcement
- Comprehensive documentation and worked examples
